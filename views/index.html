Usage

Make POST requests above to add new users and exercises. Then, you can make the following GET requests:

1) To /api/users to receive an array containing all users on the database.

/api/users

2) To /api/users/:id/logs?[from][&to][&limit] to receive the exercise log of a specific user.

/api/users/vuthe27x1tduzgr0rp10/logs?from=2021-12-01&to=2022-01-31

Note: you can add [from], [to] and [limit] as optional parameters to retrieve part of the exercise log of any user. [From] and [to] are dates in yyyy-mm-dd format while [limit] is an integer of how many logs to send back.
